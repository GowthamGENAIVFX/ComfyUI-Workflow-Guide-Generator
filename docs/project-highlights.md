# Project Highlights

## Core Features

### Workflow Type Detection

Automatically identifies:

* Flux
* SDXL
* ControlNet
* Inpainting
* Upscaling
* Video Processing
* Video to EXR

---

### Model Discovery

Automatically discovers:

* Main Models
* CLIP Models
* Qwen Models
* VAEs
* LoRAs
* ControlNets
* Upscalers

---

### Workflow Flow Generation

Converts complex workflows into simple artist-friendly stages.

Example:

Load Models
↓
Process Prompt
↓
Generate Image
↓
Save Result

---

### Dynamic Documentation Panel

Generates:

* Workflow Purpose
* Workflow Overview
* Models Used
* Main Settings
* Workflow Flow
* Output Information
* Performance Recommendations
* Key Node Explanations

---

### Artist-Friendly Design

The project prioritizes workflow understanding over technical complexity.

Documentation is written for artists rather than developers.
